# Sales & Revenue Analysis Dashboard

## Project Overview
This project analyzes sales and revenue data using Power BI.

## Features
- Revenue KPI Tracking
- Sales Quantity Analysis
- Product Count Analysis
- Revenue Trend Visualization
- Category-wise Revenue Analysis
- Region-wise Revenue Analysis
- Interactive Filters and Slicers

## Tools Used
- Power BI
- Microsoft Excel

## Files Included
- Sales_Revenue_Dashboard.pbix
- sales_data.xlsx

## Outcome
Developed an interactive dashboard to generate business insights from sales data.
